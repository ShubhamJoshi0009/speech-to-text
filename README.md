Speech to Text Converter

A simple web-based Speech to Text Converter built using HTML, CSS, and JavaScript. This application uses the Web Speech API to convert spoken words into text in real time.

Features
🎤 Start speech recognition with a single click
📝 Convert speech into text instantly
⏹️ Stop speech recognition anytime
🌐 Runs directly in the browser
📱 Responsive and easy-to-use interface
Technologies Used
HTML5
CSS3
JavaScript
Web Speech API
Font Awesome
